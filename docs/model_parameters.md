# Явные параметры модели ИК-датчика

Этот файл фиксирует параметры, которые используются в репозитории при моделировании инфракрасного датчика и при проведении экспериментальных серий. Часть параметров является базовой физической настройкой модели из `models/params.py`, часть относится к облегченной экспериментальной реализации серии 3 из `experiments/series3/common.py` и `results/experiment_*/config.json`.

## Базовая физическая модель

| Параметр | Обозначение | Значение | Источник | Комментарий |
|---|---:|---:|---|---|
| Размер активной матрицы | `resolution` | `320 x 240` пикселей | `models/params.py` | Базовое разрешение активной области сенсора: ширина x высота. |
| Полный размер с blind/boundary пикселями | `size_total` | `326 x 246` пикселей | расчет по `resolution`, `size_boundary`, `size_blind` | `320+2+2+1+1` по ширине и `240+2+2+1+1` по высоте. |
| Pitch пикселя | `pitch` | `17e-6 m = 17 um` | `models/params.py`, `experiments/series3/common.py` | Шаг между центрами соседних пикселей. |
| Размер чувствительной области пикселя | `bol_dim_h x bol_dim_v` | `17 um x 17 um` | `models/params.py` | В модели площадь пикселя равна `area = 17e-6 * 17e-6 m^2`. |
| Диапазон LWIR | `lambd` | `8e-6..14e-6 m = 8..14 um` | `models/params.py` | Диапазон интегрирования излучения черного тела. |
| Теплопроводность болометра | `G_thermal_med` | `1e-7 W/K` | `models/params.py` | Медианное значение для матрицы. |
| Теплоемкость болометра | `C_thermal_med` | `5e-9 J/K` | `models/params.py` | Медианное значение для матрицы. |
| Тепловая постоянная времени | `tau = C/G` | `0.05 s = 50 ms` | расчет по `models/params.py` | В `models/Bolometers.py` рассчитывается как `tau = C_thermal / G_thermal`. |
| Время интегрирования | `t_int` | `10e-3 s = 10 ms` | `models/params.py` | Используется в модели считывания `Readout`. |
| Частота кадров, принятая для паспорта модели | `frame_rate` | `100 Hz` | расчет `1 / t_int` | В коде нет отдельной переменной FPS; для явного описания принимается один кадр на один интервал интегрирования. |
| Разрядность АЦП | `adc_resolution`, `ADC_BITS` | `10 bit` | `models/params.py`, `experiments/series3/common.py` | Диапазон цифрового кода: `0..1023`. |
| Максимальный цифровой код | `ADC_MAX` | `1023` | `experiments/series3/common.py` | `2^10 - 1`. |
| Опорное напряжение базового ADC | `V_max` | `3.2 V` | `models/params.py` | В серии 2 дополнительно применялось калиброванное аналоговое окно. |

## NETD

В текущем репозитории NETD не задан как входная аппаратная константа. Он оценивается по результатам эксперимента 3.1 как отношение шумового стандартного отклонения к локальному наклону радиометрической характеристики:

```math
NETD = \frac{\sigma_{ADC}}{dS/dT}.
```

| Оценка NETD | Значение | Источник |
|---|---:|---|
| NETD около `300 K` | `0.359 K` | `results/experiment_01_signal_validation/metrics.csv` |
| Лучшая оценка NETD в диапазоне `280..360 K` | `0.239 K` | `results/experiment_01_signal_validation/metrics.csv`, строка `360 K` |

Для диплома корректная формулировка: в модели NETD является расчетной метрикой, а не заранее заданным паспортным параметром. В качестве численного значения можно указать `NETD ~= 0.36 K` около 300 K и лучшую полученную оценку `0.239 K` в исследованном диапазоне.

## Короткий паспорт для вставки в диплом

В работе моделируется LWIR-матрица с базовым активным разрешением `320 x 240` пикселей и шагом пикселя `17 um`. Рабочий спектральный диапазон составляет `8..14 um`. Медианные параметры болометра задают тепловую постоянную времени `tau = C/G = 0.05 s`. Время интегрирования равно `10 ms`, что соответствует принятой частоте кадров `100 Hz` при одном кадре на интервал интегрирования. АЦП имеет разрядность `10 bit`, цифровой диапазон `0..1023`. Для ускоренных экспериментов серии 3 применяются кадры `128 x 96` пикселей, а для временной динамики `112 x 80` пикселей. Базовая модель шума задается в ADC-кодах; типичные значения `sigma` составляют `2.0..3.0 ADC` в зависимости от эксперимента. NETD оценивается по радиометрической характеристике и составляет около `0.359 K` при `300 K`, а лучшая оценка в исследованном диапазоне равна `0.239 K`.
