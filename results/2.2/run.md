# Запуск 2.2 — Неравномерность болометрической матрицы и фиксированный шум FPN

## Цель
Исследовать, как разброс параметров микроболометров влияет на качество выходного сигнала.

Этот запуск должен показать вклад физических вариаций матрицы:
- сопротивления
- теплопроводности
- теплоемкости
в итоговую неравномерность кадра.

## Исследовательский вопрос
Как изменение tolerance/randomization параметров болометрической матрицы влияет
на фиксированный рисунок шума и на статистику кадра?

## Постановка
Использовать однородную сцену и фиксированную температуру, например:
300 K, 340 K, 380 K.

Для каждой температуры провести серию прогонов при нескольких уровнях tolerance:
- 1e-6
- 3e-6
- 1e-5
- 3e-5
- 1e-4

Для каждого уровня tolerance:
- сгенерировать минимум 3 реализации с разными random seed
- сохранить карты аналогового и цифрового сигнала

## Что сохранить
В папке results/2.2 сохранить:
- config.json
- fpn_metrics.csv
- seed_manifest.json
- arrays/readout_frame_tol_*_seed_*.csv
- arrays/adc_frame_tol_*_seed_*.csv
- images/readout_frame_tol_*_seed_*.png
- images/adc_frame_tol_*_seed_*.png

- images/fpn_range_vs_tolerance.png
- summary.md

## Какие метрики посчитать
Для каждого temperature / tolerance / seed:
- mean
- std
- coefficient_of_variation = std / mean
- peak_to_peak = max - min
- row_mean_std
- col_mean_std

Сводные зависимости:
- std от tolerance
- peak_to_peak от tolerance
- row/column structure от tolerance

## Что покажет запуск
Этот запуск покажет, как физический разброс параметров матрицы
превращается в fixed pattern noise.

Он позволит отделить:
- плавную неравномерность от оптики
- случайно-структурную неравномерность от болометров

## Польза для диплома
Очень полезен для объяснения, зачем вообще нужна коррекция NUC.
На его основе можно показать, что даже при одинаковой сцене
матрица без коррекции выдает неоднородный цифровой кадр.

Это один из ключевых запусков для обоснования технической части работы.
