Реализуй эксперимент 4: «Оценка эффективности фильтрации ИК-изображений перед обнаружением аномалий».

Цель:
Проверить, улучшают ли простые методы предварительной обработки качество обнаружения температурных аномалий на зашумлённых ИК-кадрах.

Что нужно сделать:
1. Использовать синтетические кадры с температурной аномалией и шумом.
2. Реализовать несколько вариантов предварительной обработки:
   - без фильтрации;
   - Gaussian blur;
   - median filter;
   - bilateral filter;
   - non-local means, если доступно без тяжёлых зависимостей.
3. Для каждого фильтра протестировать несколько параметров:
   - размер ядра;
   - sigma;
   - strength.
4. После фильтрации применить одинаковый алгоритм обнаружения.
5. Сравнить результат с истинной маской.
6. Рассчитать:
   - SNR до и после фильтрации;
   - TPR;
   - FPR;
   - Precision;
   - IoU;
   - изменение локального контраста аномалии.
7. Построить графики:
   - SNR improvement для каждого фильтра;
   - IoU для каждого фильтра;
   - TPR/FPR для каждого фильтра;
   - примеры изображений до/после фильтрации и итоговые маски.

Ожидаемый результат:
Фильтрация должна уменьшить шум и повысить устойчивость обнаружения, но слишком сильное сглаживание может размыть малые аномалии. Нужно показать компромисс между шумоподавлением и сохранением локального температурного контраста.

Требования:
- Не подбирать параметры вручную под один кадр.

- Зафиксировать seed.
- В README описать, какой фильтр оказался наиболее устойчивым и почему.
