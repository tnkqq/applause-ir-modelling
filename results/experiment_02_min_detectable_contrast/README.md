# Эксперимент 02 - минимальный обнаруживаемый температурный контраст

## Цель

Оценить, при каком температурном контрасте простая пороговая сегментация стабильно выделяет аномалию.

## Параметры

Фон 300 K; dT=[0.1, 0.2, 0.5, 1.0, 1.5, 2.0, 3.0, 4.0, 6.0, 8.0, 10.0]; формы: circle, rectangle, gaussian; seed: 3202.

## Выходные файлы

- `metrics.csv`
- `metrics_by_delta_t.csv`
- `tpr_vs_delta_t.png`
- `fpr_vs_delta_t.png`
- `iou_vs_delta_t.png`
- `detection_probability_vs_delta_t.png`
- `masks/`

## Краткий вывод

Минимальный устойчиво обнаруживаемый контраст при IoU>0.3 и вероятности >=0.9: 2 K.
