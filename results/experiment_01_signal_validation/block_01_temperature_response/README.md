# Эксперимент 01 - блок 1: температурная радиометрическая характеристика

## Цель

Проверить монотонность цифрового сигнала от температуры сцены и оценить SNR/NETD.

## Параметры

Температуры 280-360 K с шагом 10 K; кадров на уровень: 16; seed: 3101.

## Выходные файлы

- `metrics.csv`
- `metrics.json`
- `config.json`
- `signal_vs_temperature.png`
- `noise_vs_temperature.png`
- `snr_vs_temperature.png`
- `netd_estimation.png`
- `example_frames.png`

## Краткий вывод

Средний наклон радиометрической характеристики составил 7.878 ADC/K; лучшая оценка NETD равна 0.239 K; монотонность среднего сигнала: подтверждена.
