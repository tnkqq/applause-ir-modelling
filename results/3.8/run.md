Собери итоговый отчёт по всем экспериментам.

Нужно пройти по папкам results/experiment_*/ и сформировать:
1. общую таблицу summary_all_experiments.csv;
2. Markdown-отчёт results/summary_report.md;
3. папку results/figures_for_diploma/ с копиями ключевых графиков.

В summary_all_experiments.csv включить:
- номер эксперимента;
- название;
- ключевые варьируемые параметры;
- основные метрики;
- главный численный результат;
- путь к основному графику;
- краткий вывод в 1–2 предложения.

В summary_report.md для каждого эксперимента написать:
- цель;
- что моделировалось;
- какие параметры изменялись;
- какие метрики рассчитывались;
- какие графики получены;
- какой вывод можно использовать в дипломе.

Не выдумывать результаты. Использовать только сохранённые CSV/JSON из папок экспериментов.
