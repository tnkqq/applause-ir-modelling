Реализуй эксперимент 7: «Сравнение алгоритмов обнаружения температурных аномалий на синтетических ИК-кадрах».

Цель:
Сравнить несколько простых и объяснимых алгоритмов обнаружения аномалий и выбрать наиболее устойчивый вариант для практической части диплома.

Что нужно сделать:
1. Сформировать общий тестовый набор синтетических ИК-кадров:
   - разные ΔT;
   - разные размеры аномалий;
   - разные уровни шума;
   - разные положения аномалий;
   - несколько кадров без аномалий для оценки ложных срабатываний.
2. Для каждого кадра сохранить истинную маску аномалии.
3. Реализовать несколько алгоритмов:
   - глобальный порог по mean + k·std;
   - адаптивный локальный порог;
   - Otsu thresholding;
   - Difference of Gaussians / LoG;
   - простая морфологическая постобработка.
4. Для каждого алгоритма подобрать не один «лучший» порог, а протестировать диапазон параметров.
5. Рассчитать:
   - Precision;
   - Recall/TPR;
   - FPR;
   - F1-score;
   - IoU;
   - ROC/AUC, если применимо;
   - среднее время обработки одного кадра.
6. Построить:
   - ROC-кривые;
   - bar chart по F1-score;
   - bar chart по IoU;
   - таблицу времени обработки;
   - примеры успешных и неуспешных детекций.

Ожидаемый результат:
Нужно получить обоснованное сравнение алгоритмов. Ожидается, что простой глобальный порог будет работать на однородных сценах, но хуже при неоднородном фоне и шуме. Более адаптивные методы должны давать меньше ложных срабатываний, но могут быть чувствительны к параметрам.
