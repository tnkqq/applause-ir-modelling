# Явные параметры модели ИК-датчика

Основной паспорт параметров находится в `docs/model_parameters.md`; табличная версия находится в `docs/model_parameters.csv`.

Ключевые значения для диплома:

| Параметр | Значение |
|---|---:|
| Активная матрица базовой модели | `320 x 240` пикселей |
| Полная матрица с blind/boundary пикселями | `326 x 246` пикселей |
| Размер кадров большинства экспериментов серии 3 | `128 x 96` пикселей |
| Размер кадров эксперимента 3.6 | `112 x 80` пикселей |
| Pitch | `17 um` |
| LWIR-диапазон | `8..14 um` |
| Тепловая постоянная времени болометра | `tau = 0.05 s = 50 ms` |
| Время интегрирования | `10 ms` |
| Принятая частота кадров | `100 Hz` |
| ADC bits | `10 bit` |
| Диапазон ADC-кода | `0..1023` |
| Базовая sigma гауссова шума | `2.0 ADC` |
| Типичные sigma в экспериментах | `2.2..3.0 ADC` |
| NETD около 300 K | `0.359 K` |
| Лучшая NETD в диапазоне 280..360 K | `0.239 K` |

Важно: NETD в текущей реализации не является входной аппаратной константой. Она оценивается по эксперименту 3.1 как `NETD = sigma_ADC / (dS/dT)`.
