# Запуск 2.0 — базовый полноразмерный прогон

Выполнен базовый прогон однородной сцены через цепочку `Blackbody -> Optics -> Bolometers -> VectorizedReadout -> ADC`.

Параметры:
- разрешение: 160x120;
- температуры: [280, 300, 320, 340, 360, 380, 400];
- tolerance матрицы: 1e-06;
- seed: 2001;
- ADC: 10 бит, калиброванное окно `2.238774..111.940680` V.

Ключевые результаты:
- мощность центрального пикселя растет от `1.569002e-09` W до `6.994672e-09` W;
- средний ADC-код растет от `75.868` до `901.416`;
- отрицательные ADC-коды исключены за счет калиброванного analog gain/offset перед квантованием;
- доля насыщенных пикселей при максимальной температуре: `0.000000`;
- доля нулевых пикселей при минимальной температуре: `0.000000`.

Вывод: базовая цепочка дает физически интерпретируемую зависимость `температура -> мощность -> цифровой код` без аварий и без отрицательного цифрового диапазона.
