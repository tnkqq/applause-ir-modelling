# Запуск 2.0 — Базовый полноразмерный прогон однородной сцены

## Цель
Получить эталонный полноразмерный набор данных для всей цепочки моделирования:
Blackbody -> Optics -> Bolometers -> Readout -> ADC.

Этот запуск нужен как базовая точка отсчета для всех последующих экспериментов.
Он должен показать, как выглядит сигнал идеализированной однородной сцены без сильных
возмущений и с физически корректным цифровым диапазоном.

## Исследовательский вопрос
Какой вид имеет выходной сигнал инфракрасной системы при наблюдении однородного
источника излучения при разных температурах, если конвейер работает без аварий
и без чрезмерной деградации параметров матрицы?

## Постановка
Смоделировать однородную сцену с равномерной температурой.
Провести серию прогонов для температур:
280 K, 300 K, 320 K, 340 K, 360 K, 380 K, 400 K.

Размер матрицы:
- минимум 64x48
- предпочтительно 160x120
- если позволяет время вычислений — 320x240

Для каждого значения температуры сохранить промежуточные и итоговые данные:
- мощность на центральном пикселе
- карту распределения мощности по матрице
- суммарную входную мощность на пиксели
- карту аналогового сигнала после readout
- карту цифрового сигнала после ADC
- статистику кадра

## Требования к параметрам
- использовать физически разумный диапазон длин волн сенсора
- исключить очевидно некорректный режим отрицательных кодов ADC
- если нужно, ввести смещение (offset) или saturating clip перед квантованием
- зафиксировать все параметры в JSON-конфиге запуска

## Что сохранить
В папке results/2.0 сохранить:
- config.json

- optics_power_stats.csv
- adc_metrics.csv
- arrays/p_distribution_*.csv
- arrays/adc_frame_*.csv
- images/p_distribution_*.png

## Какие метрики посчитать
Для каждого кадра:
- min
- max
- mean
- std
- dynamic_range = max - min
- доля насыщенных пикселей
- доля нулевых пикселей
- центр/край отношение сигнала

## Что покажет запуск
Этот запуск даст базовую зависимость:
температура -> мощность -> аналоговый сигнал -> цифровой код.

Он покажет:
- работает ли конвейер в полном размере;
- корректен ли диапазон выходного цифрового сигнала;
- насколько сильна естественная неравномерность по полю зрения;
- есть ли насыщение при росте температуры.

## Польза для диплома
Этот запуск можно использовать в начале технической части как базовый
радиометрический и системный эксперимент.

Он дает материал для объяснения:
- закона Планка;
- in-band интегрирования;
- закона cos^4;
- перехода от оптической мощности к цифровому коду.
