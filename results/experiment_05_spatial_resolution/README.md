# Эксперимент 05 - пространственное разрешение и размер аномалии

## Цель

Показать влияние площади аномалии и коэффициента заполнения пикселя на обнаружимость.

## Параметры

Размеры: [1, 2, 4, 8, 16, 32]; fill_factor: [0.1, 0.25, 0.5, 0.75, 1.0]; dT=6 K; seed: 3505.

## Выходные файлы

- `metrics.csv`
- `metrics_grid.csv`
- `tpr_heatmap.png`
- `iou_heatmap.png`
- `min_detectable_size.png`
- `size_examples.png`

## Краткий вывод

Минимальный обнаруживаемый размер при TPR>0.9 и FPR<0.01: 1 px; малый fill_factor заметно ухудшает обнаружение.
