# Запуск 2.5 — Интеграционный прогон с температурной аномалией

## Цель
Провести итоговый прикладной эксперимент, близкий к задаче диплома:
смоделировать сцену с локальной температурной аномалией
и оценить, насколько она различима на выходе системы.

## Исследовательский вопрос
Как меняется наблюдаемость температурной аномалии при изменении:
- температуры аномалии,
- размера аномалии,
- уровня неравномерности,
- наличия/отсутствия NUC-коррекции?

## Постановка
Сформировать сцену:
- фон: 300 K
- аномалия: круг или прямоугольная область

Серия температур аномалии:
- 320 K
- 340 K
- 360 K
- 400 K

Серия размеров аномалии:
- малая
- средняя
- крупная

Для каждой комбинации:
- без коррекции
- после NUC-коррекции

Если возможно, сделать также два уровня tolerance матрицы:
- низкий
- средний

## Что сохранить
В папке results/2.5 сохранить:
- config.json
- anomaly_metrics.csv
- arrays/scene_map_*.csv
- arrays/adc_uncorrected_*.csv
- arrays/adc_corrected_*.csv
- images/scene_map_*.png
- images/adc_uncorrected_*.png
- images/adc_corrected_*.png
- images/anomaly_contrast_vs_temperature.png
- images/anomaly_visibility_vs_size.png
- summary.md

## Какие метрики посчитать
- mean_anomaly
- mean_background
- contrast = mean_anomaly - mean_background
- contrast_ratio = mean_anomaly / mean_background
- std_background
- snr_like = contrast / std_background
- оценка площади обнаруживаемой аномалии
- доля пикселей аномалии, сохранивших контраст после всех этапов

## Что покажет запуск
Этот запуск впервые связывает всю модель с прикладной задачей обнаружения
температурных аномалий.

Он покажет:
- какой температурный контраст реально виден после прохождения через всю систему;
- как ухудшают ситуацию неравномерность и дефекты;
- насколько помогает NUC-коррекция;
- какие аномалии система различает уверенно, а какие — плохо.

## Польза для диплома
Это главный запуск для технической части.
Он соединяет математическую модель ИК-датчика с конечной прикладной целью:
обнаружением температурной аномалии.

Именно этот запуск лучше всего использовать для итоговых графиков,
таблиц и инженерных выводов.
